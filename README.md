# utm-catcher
Saving utm parameters to a cookie.

## Using

Include:
```html
<script src="https://cdn.jsdelivr.net/gh/Zeboton/utm-catcher/utm-catcher.min.js" />
```

Call with jQuery:
```javascript
$(function () {
    lp.fn.setUTM();
});
```

Call with plain JavaScript:
```javascript
lp.fn.setUTM();
```
